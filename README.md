An online store for beautiful handmade embroidery and paintings by Mrs. Ranjula Kumari.

## Features

- Browse handmade products by category
- User registration and login
- Shopping cart
- Order via Cash on Delivery or WhatsApp
- Admin panel for managing products and orders

## How to Use

### For Customers
1. Visit the website
2. Browse products by clicking on categories
3. Register/Login to add items to cart
4. Place order with Cash on Delivery or WhatsApp

### For Admin
1. Go to `/admin` page
2. Login with admin credentials
3. Manage products, orders, categories, and messages

## Quick Start (For Developers)

```bash
# Install dependencies
npm install

# Seed sample data
npm run seed

# Start server
npm start
```

Visit `http://localhost:3000`

## Environment Variables

Create a `.env` file:
```
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=3000
```

## Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
